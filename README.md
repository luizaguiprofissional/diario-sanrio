# 🎀 Diário Sanrio

Um aplicativo de anotações e organização super fofo e mágico, inspirado no universo Sanrio. Organize suas ideias, tarefas e segredos em um ambiente acolhedor e cheio de estilo!

## ✨ Sobre o Projeto

O **Diário Sanrio** foi criado para ser o seu cantinho especial na web. Com uma estética *Kawaii*, ele combina a doçura dos personagens clássicos com funcionalidades modernas de produtividade.

### Principais Funcionalidades:
- 📝 **Anotações Coloridas**: Crie notas com títulos e conteúdos organizados.
- 📂 **Categorias**: Organize-se por temas como Receitas, Dev, Compras, Pessoal, Lazer, Ideias e Estudos.
- ✅ **Listas de Tarefas (Todos)**: Adicione sub-tarefas dentro de cada nota para acompanhar seu progresso.
- ❤️ **Favoritos**: Marque suas notas mais importantes com um coração.
- 🔐 **Autenticação**: Seu diário é protegido por login e senha.
- 🎨 **Personalização**: Altere seu avatar e nome de perfil.

## 🚀 Tecnologias Utilizadas

Este projeto utiliza ferramentas modernas do ecossistema JavaScript/TypeScript:

- **Core**: [React 19](https://react.dev/) & [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite 6](https://vitejs.dev/)
- **Backend**: [Express 4](https://expressjs.com/) (Node.js)
- **Banco de Dados**: [SQLite](https://www.sqlite.org/) (via `better-sqlite3`)
- **Estilização**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animações**: [Motion](https://motion.dev/) (Framer Motion)
- **Ícones**: [Lucide React](https://lucide.dev/)

## 🛠️ Como Iniciar

Siga os passos abaixo para rodar o projeto localmente em sua máquina:

### Pré-requisitos
- [Node.js](https://nodejs.org/) instalado.
- [npm](https://www.npmjs.com/) ou outro gerenciador de pacotes.

### Passo a Passo

1. **Instalar Dependências**:
   Abra o terminal na pasta do projeto e execute:
   ```bash
   npm install
   ```

2. **Configuração do Ambiente**:
   O projeto já vem com um arquivo `.env.example`. Para o funcionamento básico, você pode apenas copiá-lo:
   ```bash
   copy .env.example .env
   ```

3. **Iniciar o Servidor de Desenvolvimento**:
   Execute o comando abaixo para iniciar tanto o backend quanto o frontend (Vite):
   ```bash
   npm run dev
   ```

4. **Acessar o Aplicativo**:
   Abra o seu navegador e vá para:
   [http://localhost:3000](http://localhost:3000)
